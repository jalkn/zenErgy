# ⚡ BIORUSH // Functional Food Core

E-commerce minimalista y de alta conversión optimizado para entornos móviles. Distribución y despliegue del catálogo oficial de alimentos funcionales y nootrópicos adaptógenos desde Santa Elena, Antioquia.

---

## 💎 Design Philosophy: Conversion & Frictionless
* **Minimalismo Radical:** Interfaz de pantalla única (Single Page View) diseñada bajo estética de terminal cibernética, eliminando carritos de compra tradicionales para evitar el abandono de flujo.
* **Haptic UI:** Integración de API de vibración nativa (`navigator.vibrate`) y eventos interactivos táctiles para simular la manipulación física de empaques (`Pointer Events`).
* **WhatsApp Checkout Loop:** Los estados del catálogo se consolidan en un único payload de metadatos encriptado textualmente, derivando el cierre de la venta directamente al canal de mensajería instantánea.

---

## 🛠️ Stack & Architecture
* **Frontend:** HTML5 semántico, Tailwind CSS (Configuración nativa extendida).
* **Interacciones:** Vanilla JavaScript optimizado para renderizado asíncrono y manipulación del DOM sin frameworks pesados.
* **Despliegue:** CDN global optimizado para latencia cero en redes móviles latinoamericanas.
